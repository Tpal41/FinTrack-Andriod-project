# 📱 FinTrack App - Visual Overview

## 🎨 **Design Theme**

### Color Palette
- **Primary**: Emerald Green (#20E3A2) - For positive actions, income
- **Secondary**: Gold (#FFC857) - For highlights, warnings
- **Accent**: Coral Red (#FF6B6B) - For expenses, alerts
- **Background Dark**: Deep Navy (#020713, #050B18)
- **Background Light**: Clean White (#F8F9FB)
- **Glass Effect**: Transparent cards with blur and gradient borders

### UI Style
- **Glassmorphic Design** - Frosted glass effect cards with soft shadows
- **Premium Typography** - Bold headlines with gradient text effects
- **Smooth Animations** - Compose-based transitions
- **Edge-to-Edge** - Immersive fullscreen experience

## 🎯 **Key Design Features**

### Glassmorphism Effect
- Semi-transparent backgrounds with blur
- Gradient borders (white to black fade)
- Soft shadows for depth
- Premium feel

### Color Coding
- 🟢 **Green (Emerald)**: Income, savings, positive
- 🟡 **Gold**: Warnings, highlights, health index
- 🔴 **Red (Coral)**: Expenses, exceeded budgets, delete actions
- 🔵 **Navy**: Dark mode background

### Typography
- **Headlines**: Bold, gradient text effects
- **Body**: Clean, readable
- **Numbers**: Extra bold for emphasis

### Interactions
- Haptic feedback on important actions
- Smooth compose animations
- Pull-to-refresh
- Swipe gestures
- FAB for quick add

## 🎨 **Dark vs Light Mode**

### Dark Mode (Default)
- Background: Deep Navy (#020713)
- Cards: Transparent glass with blur
- Text: White (#F6F8FB)
- Accent: Emerald green

### Light Mode
- Background: Clean White (#F8F9FB)
- Cards: White with subtle shadows
- Text: Dark (#1E1E2C)
- Accent: Emerald green (same)
